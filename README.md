# ICEI Research Repository

## Overview

This repository contains all resources for the IEEE research project, including manuscript drafts, source code, datasets, literature review, experimental results, and project documentation. Team Members - KunalSingh Thakur, Nikhil Sharma.

## Repository Structure

```
paper/              IEEE manuscript and bibliography
code/               Source code
datasets/           Raw and processed datasets
figures/            Figures used in the manuscript
results/            Experimental outputs
literature/         Research papers and notes
meeting-notes/      Meeting documentation
scripts/            Utility scripts
```

## Version Control

This project follows Conventional Commits for maintaining a clean and traceable Git history.
